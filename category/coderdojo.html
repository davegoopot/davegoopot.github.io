Phaser.js at the Manchester CoderDojo

Published: Mon 14 November 2016
By Dave Potts

In CoderDojo.

At this month's Manchester CoderDojo I spent the time working with the participants going through the first tutorial using the Javascript 2-D game engine Phaser.js.

It was just about on the edge of what the kids could keep up with but I'm optimistic that if they stick with it we'll be able to make some interesting games.

Proudly powered by Pelican, which takes great advantage of Python.
