Getting Started With Scuttlebutt

Published: Sat 29 July 2017
By Dave Potts

In Web Dev.

I've set myself up on Scuttlebutt a decentralised social network. I heard about it as an off-grid Facebook alternative from this Software Engineering Daily podcast.

If you'd like to connect with me on Scuttlebug I'm @jnA0/zkHb0TYLheAhmzNxK6MlOWqHlsk+xxMdy/pRDY=.ed25519

Here's how I set it up on Windows 10:

  1. Download the windows installer from the GitHub repo. I got version 3.6.6
  2. Install on a Windows 10 machine. I allowed the network access through the firewall when asked
  3. I set up my profile and published
  4. Not really knowing how things worked I randomly picked http://ssb.rootsystems.nz/ to ask for an invite code
  5. I chose "Join Pub" in patchwork and pasted in my invite code
  6. That joined just fine and lots of channels started appearing
  7. I subscribed to the #new-people channel
  8. It took a while for all the messages to download
  9. While waiting for the messages to download I also subscribed to the #python channel
  10. After about 5 minutes I was up and running and could post to the network
