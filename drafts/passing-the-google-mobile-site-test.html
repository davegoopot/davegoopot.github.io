Passing the Google Mobile Site Test

Published: Mon 08 May 2017
By Dave Potts

In Web Dev.

This blog site didn't pass the Google mobile friencly site test. Here's what I did to fix it:

  1. Fix the Viewport

The viewport is just a HTML meta tag that tells small screen devices how they should scale pages. See the W3 Schools entry on the viewport. To change the viewport for my blog site I had to update the template entry in my Pelican theme. To update the theme I copied the standard "notmyidea" theme from the site package and updated the base.html file to include a suitable viewport metatag. Finally I updated the Pelican build files to use my update theme and republished.

Proudly powered by Pelican, which takes great advantage of Python.
